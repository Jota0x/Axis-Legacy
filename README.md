# Axis: Seu Centro de Performance
O **Axis** é um ecossistema completo de saúde e bem-estar em desenvolvimento. O objetivo é oferecer ao usuário um gerenciador inteligente de dieta e planos de treino.

Este projeto está sendo desenvolvido em **Java** com o framework **Spring Boot**, focando em boas práticas de Programação Orientada a Objetos (POO).

## Status do Projeto: Em Construção

Atualmente, o projeto está na fase de estruturação da lógica de negócio e modelos de dados.
